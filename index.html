What is Nuverba?

Nuverba is a specific website for word and character counting with the ability to use filters if required.


It's was build with html css and javascript , at the moment the page only supports plain text but in the future I plan to add pdfs, word files and even photos if the text can be clearly identified


If you are a student, a teacher or another type of professional or even a simple curious person, this tool will be very useful.
